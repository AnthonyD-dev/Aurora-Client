# Aurora

A modern Windows utility built with C# WinForms and Siticone UI.

Aurora provides a clean, lightweight interface focused on performance, customization, and ease of use.

![Aurora Banner](Resources/aurorabg.png)

## Features

* Modern Muted Crimson UI
* Custom Notification System
* Activity Logging
* Aurora Splash Screen
* Process Detection
* Plugin Management
* Aurora Helper
* Automatic Updates
* Lightweight & Fast
